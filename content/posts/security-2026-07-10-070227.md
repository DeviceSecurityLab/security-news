### [Microsoft patches RoguePlanet Defender zero-day vulnerability](https://www.bleepingcomputer.com/news/microsoft/microsoft-patches-rogueplanet-defender-zero-day-vulnerability/)
**2026-07-09 14:42 JST ／ BleepingComputer ／ HIGH**


**要約:** Microsoft は、Microsoft Defender におけるゼロデイ脆弱性 CVE-2026-50656（通称 RoguePlanet）のパッチをリリースした。本脆弱性はレースコンディションに起因し、完全にパッチ適用済みの Windows 10 / Windows 11 環境において、リアルタイム保護が有効な状態でも攻撃者が SYSTEM 権限でコマンドプロンプトを起動することを可能にする。脆弱性を発見した研究者 Nightmare Eclipse は報奨金プログラムを巡る紛争の中で詳細を公開。修正は Microsoft Malware Protection Engine バージョン 1.1.26060.3008 として提供された。


**なぜ重要か:** Microsoft Defender は Windows 標準搭載のセキュリティ製品であり、事実上すべての Windows 環境に影響する。SYSTEM 権限の奪取は完全なシステム制御を意味し、悪用された場合の影響は極めて大きい。


**影響:** すべての Windows 10 / Windows 11 システムが対象。攻撃者は Defender の防御を回避し、完全なシステム制御を獲得可能。


**次のアクション:** Microsoft Malware Protection Engine が自動更新されることを確認。即座に Windows Update を適用し、バージョン 1.1.26060.3008 以上であることを検証すること。
