### [Check Point links VPN zero-day attacks to Qilin ransomware gang](https://www.bleepingcomputer.com/news/security/check-point-links-vpn-zero-day-attacks-to-qilin-ransomware-gang/)
**2026-06-08 22:05 JST ／ BleepingComputer ／ HIGH**

**要約:** Check PointはRemote Access VPNおよびMobile Access製品に存在する重大なゼロデイ脆弱性（CVE-2026-50751）を開示した。2026年5月7日から悪用が確認されており、非認証のリモート攻撃者が非推奨のIKEv1プロトコルを使用するVPNの認証をバイパスできる。6月初旬に攻撃が急増し、世界数十の組織に影響。少なくとも1件はQilinランサムウェア関連グループによる攻撃と確認されている。さらにIKEv1証明書検証に関する2つ目の脆弱性（CVE-2026-50752）も発見された。

**なぜ重要か:** 世界的に広く展開されているCheck Point VPN製品に影響する重大なゼロデイであり、実際の攻撃が確認され、Qilinランサムウェアとの関連性も指摘されている。

**影響:** Check Point Security GatewayをIKEv1で運用している全組織。数十の組織ですでに侵害が確認済み。

**次のアクション:** IKEv1を無効化しIKEv2を強制。マシン証明書を必須化しIPSを有効化。CISA KEVにも登録済みのため、2026年6月11日までの対応が推奨される。

### [Gogs patches critical zero-day enabling remote code execution](https://www.bleepingcomputer.com/news/security/gogs-patches-critical-zero-day-enabling-remote-code-execution/)
**2026-06-09 01:18 JST ／ BleepingComputer ／ HIGH**
