### [Ghost Accounts Abuse GitHub API in Mass Recon Campaign](https://www.securityweek.com/ghost-accounts-abuse-github-api-in-mass-recon-campaign/)
**2026-07-12 02:30 JST ／ SecurityWeek ／ LOW**

**要約:** 複数のキャンペーンがゴーストアカウントを使用してGitHub組織のリポジトリやメンバーをマッピングする大規模な偵察活動を行っている。

**なぜ重要か:** GitHub APIを悪用した組織的な偵察は、サプライチェーン攻撃や標的型攻撃の事前準備段階である可能性が高く、多くの企業が無防備な状態でGitHub上に情報を公開している。

**影響:** GitHub上に組織情報を公開している企業は、攻撃者に組織構造やリポジトリ情報を収集されるリスクがある。

**次のアクション:** GitHub組織の公開リポジトリ設定やメンバー情報の可視性設定を見直す。不要な公開情報は制限し、Organizationのセキュリティ設定を監査する。

### [Compromised jscrambler 8.14.0 npm Release Drops Rust Infostealer During Install](https://thehackernews.com/2026/07/compromised-jscrambler-8140-npm-release.html)
**2026-07-12 02:59 JST ／ The Hacker News ／ LOW**

**要約:** jscrambler npmパッケージのバージョン8.14.0がサプライチェーン攻撃により改ざんされ、preinstallフック経由でRust製の情報窃取マルウェア（Infostealer）を実行する。Windows、macOS、Linuxの3プラットフォーム向けのネイティブバイナリが含まれていた。

**なぜ重要か:** jscramblerはJavaScriptコードの難読化ツールとして広く利用されており、サプライチェーンを通じて多数のプロジェクトに影響を及ぼす可能性がある。Socketにより公開6分後に検出されたが、既に影響を受けた環境がある。

**影響:** 当該バージョンをインストールした環境で認証情報や機密情報が窃取される可能性がある。

**次のアクション:** jscrambler 8.14.0をインストールしている環境を調査し、直ちにパージする。パッケージロックファイルを確認し、preinstallスクリプトの実行ログを監査する。影響を受けた可能性のある認証情報をローテーションする。

### [Australia warns of global campaign targeting vulnerable CMS platforms](https://www.bleepingcomputer.com/news/security/australia-warns-of-global-campaign-targeting-vulnerable-cms-platforms/)
**2026-07-11 23:18 JST ／ BleepingComputer ／ LOW**

**要約:** オーストラリアサイバーセキュリティセンター（ACSC）が、脆弱なCMSプラットフォームとプラグインを標的とした世界的な悪用キャンペーンについて警告を発出した。

**なぜ重要か:** 政府機関による公式警告であり、実際に広範囲で悪用が進行中であることを示している。CMSは多くの組織で運用されており、パッチ未適用のプラグインが攻撃経路となる。

**影響:** 脆弱なCMSやプラグインを運用する組織は、Webサイトの改ざんやデータ漏洩のリスクにさらされる。

**次のアクション:** 使用中のCMSと全プラグインのバージョンを確認し、最新のセキュリティパッチを適用する。ACSCの勧告に従い、不要なプラグインの削除とWAFの強化を検討する。
