## 脆弱性

### [cPanel, WHM Release Fixes for Three New Vulnerabilities — Patch Now](https://thehackernews.com/2026/05/cpanel-whm-patch-3-new-vulnerabilities.html)

**2026-05-09 16:16 JST ／ The Hacker News ／ LOW**


**要約:** cPanelがcPanelおよびWeb Host Manager (WHM)の3つの脆弱性に対するアップデートをリリースした。CVE-2026-29201 (CVSS 4.3) は「feature::LOADFEATUREFILE」adminbin呼び出しにおける入力検証不足で、特権昇格・コード実行・サービス妨害 (DoS) に悪用される可能性がある。


**なぜ重要か:** cPanel/WHMは世界中の大多数のWebホスティング事業者が採用する管理プラットフォームであり、悪用されれば複数の顧客サイトを一括で侵害できる。


**影響:** cPanel/WHMを利用するホスティング事業者および顧客環境が特権昇格・コード実行のリスクにさらされる。


**次のアクション:** cPanel/WHMを最新バージョンへ直ちにアップデートする。適用前に変更管理プロセスを経る場合も、対応を最優先で計画すること。

## 事故・インシデント

### [JDownloader site hacked to replace installers with Python RAT malware](https://www.bleepingcomputer.com/news/security/jdownloader-site-hacked-to-replace-installers-with-python-rat-malware/)

**2026-05-10 04:27 JST ／ BleepingComputer ／ LOW**


**要約:** 人気ダウンロードマネージャー JDownloader の公式サイトが今週初めに侵害され、WindowsおよびLinux向けの悪意あるインストーラーが配布された。Windowsペイロードには Python ベースのリモートアクセス型トロイの木馬 (RAT) が含まれていた。


**なぜ重要か:** 公式配布チャネル自体が改ざんされたサプライチェーン攻撃であり、正規の手順でダウンロードしたユーザーがそのままマルウェア感染するため、エンドポイントのウイルス対策だけでは検知が難しい。


**影響:** 侵害期間中に JDownloader のインストーラーをダウンロードした Windows・Linux ユーザーが RAT 感染の被害を受けた可能性がある。感染した場合、攻撃者による遠隔操作・データ窃取・横展開のリスクがある。


**次のアクション:** 直近で JDownloader をインストール・更新したユーザーはシステムをスキャンし、公式の修正済みインストーラーで再インストールする。組織内での利用状況を確認し、感染端末の隔離と侵害調査を実施する。

### [Fake OpenAI repository on Hugging Face pushes infostealer malware](https://www.bleepingcomputer.com/news/security/fake-openai-repository-on-hugging-face-pushes-infostealer-malware/)

**2026-05-09 23:26 JST ／ BleepingComputer ／ LOW**


**要約:** Hugging Face のトレンドリストに掲載された偽 OpenAI「Privacy Filter」リポジトリが、Windowsユーザーに情報窃取マルウェア (Infostealer) を配布していた。著名AIブランドになりすました悪意あるリポジトリが高い露出度を獲得した事例。


**なぜ重要か:** AI 研究者・開発者が多く利用する Hugging Face という信頼されたプラットフォームを悪用し、OpenAI というブランドを偽装した点で欺瞞性が高く、技術者であっても被害に遭いやすい。


**影響:** 偽リポジトリを通じてマルウェアをダウンロードした Windows ユーザーは、認証情報・セッショントークン・機密データが窃取された可能性がある。


**次のアクション:** Hugging Face からダウンロードしたファイルの発行元をリポジトリの所有者情報で確認する。組織内のダウンロード履歴を調査し、不審なプロセス・認証情報漏洩の兆候を確認する。
